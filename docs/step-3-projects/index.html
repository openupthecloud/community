Step 3 - Projects

To work on a project, you'll want to understand both Git (the technology) and GitHub (the company / service).

Git / GitHub

Practice:

  1. Making a GitHub Account
  2. Making a repo on your own account
  3. Making + commenting an issue
  4. Raising a pull request on your own repo

Resources:

  • Git and GitHub for Beginners - Crash Course

Docker

Depending on the project you're building

Practice:

  1. Installing the docker CLI on your machine
  2. Pulling a docker image from DockerHub
  3. Writing your own Dockerfile

Resources:

  • Docker Tutorial for Beginners - A Full DevOps Course on How to Run Applications in Containers
  • Learn To Cloud Guide - Phase 4: DevOps Fundamentals

Shell scripting

Practice:

  1. Writing a short shell script
  2. Try to: print a strings
  3. Try to: do basic arithmatic, e.g. addition, subtraction

Resources:

  • Learn To Cloud Guide - Phase 1: Linux Bash and Networking fundamentals
  • Bash scripting will change your life
